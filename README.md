## Descripcion general del sistema

El sistema desarrollado es una aplicacion de consola para administrar una biblioteca. Permite registrar usuarios, registrar libros, realizar prestamos, devolver libros y calcular multas por dias de retraso.

El programa utiliza un menu principal con distintas opciones de interaccion. Los datos se guardan durante la ejecucion en estructuras tipo diccionario, donde se almacenan los usuarios, los libros disponibles y las estadisticas generales del sistema.

Tambien permite consultar estadisticas, como la cantidad total de prestamos realizados, la recaudacion acumulada por multas y la cantidad de veces que fue prestado cada libro.

## Instrucciones de ejecucion

1. Abrir una terminal en la carpeta del proyecto.
2. Seleccionar una opcion del menu ingresando un numero del 1 al 7.
3. Seguir las instrucciones que muestra el sistema por consola.
